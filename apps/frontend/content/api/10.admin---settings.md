# /v1/admin/settings

## GET `/v1/admin/settings/`

**Permissions:** `Authenticated`

Get all application settings.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/admin/settings/', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/admin/settings/ \
  -H "Authorization: Bearer YOUR_TOKEN"
```

---

## POST `/v1/admin/settings/`

**Permissions:** `Admin`

Update application settings.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `data` | `object` | Yes | Payload data. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/admin/settings/', { /* payload */ }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/admin/settings/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
