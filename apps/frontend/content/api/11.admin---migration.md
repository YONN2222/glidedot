# /v1/admin/migration

## GET `/v1/admin/migration/backup`

**Permissions:** `Authenticated`

Download a backup of the entire database as a ZIP file.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/admin/migration/backup', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/admin/migration/backup \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## POST `/v1/admin/migration/backup`

**Permissions:** `Authenticated`

Restore database from a backup ZIP file.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `data` | `object` | Yes | Payload data. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/admin/migration/backup', { /* payload */ }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/admin/migration/backup \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
