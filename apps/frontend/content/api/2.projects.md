# Projects

Discover the localization projects you have access to.

## List Projects
Retrieve a list of all projects available to your API key. You will need the `id` of a project to query its languages or fetch its translations.

### Request
`GET /localization/projects`

**Headers:** `Authorization: Bearer <API_KEY>`

### Example Response
```json
[
  {
    "id": 1,
    "name": "Web Application",
    "sourceLanguageId": 1
  },
  {
    "id": 2,
    "name": "iOS Mobile App",
    "sourceLanguageId": 2
  }
]
```
