# /v1/sessions

## POST `/v1/sessions/login`

**Permissions:** `None`

Authenticate using username and password. Returns user object.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `username` | `string` | Yes | The user's username. |
| `password` | `string` | Yes | The user's password. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/sessions/login', { username: 'admin', password: 'secure123' }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/sessions/login \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```

## POST `/v1/sessions/login/oidc`

**Permissions:** `None`

Authenticate using OIDC details. Returns user object.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `email` | `string` | Yes | The OIDC email address. |
| `username` | `string` | Yes | The OIDC username. |
| `avatarUrl` | `string` | No | The OIDC profile picture URL. |
| `groups` | `array` | No | List of OIDC groups. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/sessions/login/oidc', { email: 'user@example.com', username: 'user', avatarUrl: '...', groups: [] }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/sessions/login/oidc \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
