# /v1/setup

## GET `/v1/setup/status`

**Permissions:** `None`

Checks if initial setup is required.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/setup/status', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/setup/status \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## POST `/v1/setup/initial`

**Permissions:** `None`

Performs initial setup if required.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `username` | `string` | Yes | The admin username. |
| `email` | `string` | Yes | The admin email. |
| `password` | `string` | Yes | The admin password. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/setup/initial', { username: 'admin', email: 'admin@example.com', password: 'secure123' }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/setup/initial \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
