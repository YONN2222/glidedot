# /v1/localization/labels

## GET `/v1/localization/labels/:projectId`

**Permissions:** `Project Access`

Get all labels for a project.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/localization/labels/:projectId', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/localization/labels/:projectId \
  -H "Authorization: Bearer YOUR_TOKEN"
```
