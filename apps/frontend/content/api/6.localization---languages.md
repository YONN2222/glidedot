# /v1/localization/languages

## GET `/v1/localization/languages/`

**Permissions:** `Authenticated`

Get all available languages.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/localization/languages/', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/localization/languages/ \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## POST `/v1/localization/languages/`

**Permissions:** `Admin`

Create a new language.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `data` | `object` | Yes | Payload data. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/localization/languages/', { /* payload */ }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/localization/languages/ \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
