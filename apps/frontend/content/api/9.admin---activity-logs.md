# /v1/admin/activity-logs

## GET `/v1/admin/activity-logs`

**Permissions:** `Admin`

Get system activity logs.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/admin/activity-logs', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/admin/activity-logs \
  -H "Authorization: Bearer YOUR_TOKEN"
```
