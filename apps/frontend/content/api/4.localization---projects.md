# /v1/localization/projects

## GET `/v1/localization/projects/`

**Permissions:** `Authenticated`

Get all projects accessible to the user.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/localization/projects/', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/localization/projects/ \
  -H "Authorization: Bearer YOUR_TOKEN"
```
