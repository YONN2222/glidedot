# /v1/localization/keys

## GET `/v1/localization/keys/:projectId`

**Permissions:** `Project Access`

Get all keys for a project.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/localization/keys/:projectId', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/localization/keys/:projectId \
  -H "Authorization: Bearer YOUR_TOKEN"
```
