# /v1/users

## GET `/v1/users`

**Permissions:** `Admin`

Get all users.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/users', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/users \
  -H "Authorization: Bearer YOUR_TOKEN"
```

---

## POST `/v1/users`

**Permissions:** `Admin`

Create a new user.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `username` | `string` | Yes | The new user's username. |
| `email` | `string` | Yes | The new user's email. |
| `password` | `string` | Yes | The new user's password. |
| `isAdmin` | `boolean` | No | Whether the user is an admin. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/users', { username: 'johndoe', email: 'john@example.com', password: 'secure123', isAdmin: false }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/users \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```

## PATCH `/v1/users/me`

**Permissions:** `Authenticated`

Update current user profile.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `email` | `string` | No | Update email address. |
| `avatarUrl` | `string` | No | Update profile picture URL. |
| `password` | `string` | No | New password (requires `oldPassword`). |
| `oldPassword` | `string` | No | Current password, required if changing password. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.patch('https://api.yourdomain.com/v1/users/me', { email: 'new@example.com', avatarUrl: 'https://...', password: 'new', oldPassword: 'old' }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X PATCH https://api.yourdomain.com/v1/users/me \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
