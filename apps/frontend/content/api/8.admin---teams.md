# /v1/admin/teams

## GET `/v1/admin/teams`

**Permissions:** `Admin`

Get all teams.

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.get('https://api.yourdomain.com/v1/admin/teams', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X GET https://api.yourdomain.com/v1/admin/teams \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## POST `/v1/admin/teams`

**Permissions:** `Admin`

Create a new team.

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | `string` | Yes | The name of the new team. |

### Examples

#### TypeScript

```ts
import axios from 'axios';

const response = await axios.post('https://api.yourdomain.com/v1/admin/teams', { name: 'Development Team' }, {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = response.data;
```

#### cURL

```bash
curl -X POST https://api.yourdomain.com/v1/admin/teams \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{ /* payload */ }'
```
